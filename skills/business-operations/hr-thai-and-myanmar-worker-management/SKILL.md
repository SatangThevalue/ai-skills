## 1. กฎหมายแรงงานพื้นฐาน

### 1.1 กฎหมายหลัก
- **พระราชบัญญัติแรงงาน พ.ศ. 2541** (แก้ไข พ.ศ. 2567)
- **ประกาศกระทรวงแรงงาน เรื่อง การจัดการแรงงานต่างด้าว พ.ศ. 2561**
- **พระราชกฤษฎีกา การอนุญาตทำงานชนต่างด้าว พ.ศ. 2522** (อัพเดท พ.ศ. 2567)

### 1.2 สัดส่วนพนักงานไทยต่อชาวต่างชาติ
- **อัตรามาตรฐาน**: พนักงานไทย 4 : 1 ชาวต่างชาติ
- **กรณียกเว้น**:
  - โครงการอุตสาหกรรมเฉพาะโรงพยาบาล (โรงพยาบาล, โรงแรมระดับ 5 ดาวขึ้นไป)
  - สถานศึกษา
  - โครงการ BOI (กิจการพิเศษ)
  - บริษัทจดทะเบียนในตลาดหลักทรัพย์

## 2. การจ้างแรงงานต่างชาติ

### 2.1 อาชีพที่ห้ามคนต่างด้าวทำ
**ตามพระราชกฤษฎีกา การอนุญาตทำงานชนต่างด้าว พ.ศ. 2522 (แก้ไข)**

| หมวด | อาชีพที่ห้าม |
|-------|----------------|
| 1 | งานกรรมกร งานกสิกรรรม งานเลี้ยงสัตว์ งานป่าไม้ งานไฟฟ้า งานซ่อมแซมยานพาหนะ |
| 2 | งานพลาดพล่าด งานขายของ งานโรงแรม (โดยมิได้เป็นฝ่ายขาย) |
| 3 | งานบริหารธุรกิจขาย การตลาด งานสื่อสารประชาสัมพันธ์ งานวิเคราะห์ระบบคอมพิวเตอร์ |
| 4 | งานสร้างสรรค์ ศิลปะวัฒนธรรม งานวิจัยพัฒนา |
| 5 | งานธุรกรรมทางการเงินการธนาคาร งานที่ต้องใช้ใบอนุญาตหรือรับรองการฝึกอบรมพิเศษ |

### 2.2 การขอใบอนุญาตทำงานแรงงานต่างด้าว

#### ขั้นตอนการขอใบอนุญาต (สำหรับนายจ้าง)

```bash
# ขั้นตอนที่ 1: จัดเตรียมเอกสาร
# - สัญญาการจ้างแรงงานต่างด้าว
# - สำเนาใบอนุญาตทำงานของแรงงานต่างด้าว
# - สำเนาบัตรประจำตัวคนซึ่งไม่มีสัญชาติไทย (บัตรชมพู)
# - รายการคุณสมบัติพนักงานไทยปัจจุบัน

# ขั้นตอนที่ 2: ยื่นคำร้อง
# ผ่านระบบออนไลน์: https://alien.mol.go.th/

# ขั้นตอนที่ 3: รออนุมัติ (ปกติ 7-15 วันทำการ)

# ขั้นตอนที่ 4: แจ้งข้อมูลให้แรงงานต่างด้าว
# - ใบอนุญาตทำงาน
# - บัตรประชาชน (Thai Smart Card)
# - เลขประจำตัวประชาชน
```

### 2.3 ค่าติดตั้ง (Fees)

| ประเภท | ค่าธรรมเนียม |
|---------|----------------|
| ใบอนุญาตทำงาน | 500 บาท/คน |
| ใบอนุญาตทำงาน (ต่ออายุ) | 300 บาท/คน |
| ค่านายหน้า | 10,000 - 30,000 บาท (ขึ้นกับบริษัท) |

## 3. ระบบเงินเดือนและกฎหมาย

### 3.1 เงื่อนไขพื้นฐาน

**ความแตกต่างระหว่างพนักงานไทยกับแรงงานต่างชาติ**

| รายการ | พนักงานไทย | แรงงานต่างชาติ |
|--------|-------------|------------------|
| เงินเดือนขั้นต่ำ | 14,000 บาท (อัตราและเงื่อนไขพิเศษ) | ไม่เรียกว่า "ขั้นต่ำ" แต่ต้องไม่ต่ำกว่าคนไทย |
| เงินชดเชยวันละ | 300 บาท/วัน | 300 บาท/วัน |
| เวลาทำงาน | 8 ชม./วัน | 8 ชม./วัน |
| วันหยุดสะสม | 8% ของเงินเดือน | 8% ของเงินเดือน |

### 3.2 ระบบเงินโบนัส

**โบนัสประจำปี (ตามสัญญา)**

| สถานะ | โบนัส |
|---------|--------|
| คนไทย | 1-2 เดือน (ขึ้นกับบริษัท) |
| คนต่างชาติ | 1 เดือน (ขั้นต่ำ) |

**โบนัสพิเศษ**
- คนต่างชาติมีสิทธิ์ได้รับโบนัสพิเศษ หากได้รับการจ้างผ่านนายหน้า (Manpower)
- ควรกำหนดในสัญญาและสะสมเป็นใบเสร็จรับเงิน

## 4. ประกันสังคม

### 4.1 การเข้าระบบประกันสังคมของแรงงานต่างด้าว

**สิทธิประโยชน์ที่เทียบเท่าคนไทย**
- เงินชดเชยเจ็บป่วย
- เงินชดเชยคลอดบุตร
- เงินชดเชยชราภาพ/ว่างงาน
- เงินออมแห่งชาติ (กองทุน)

### 4.2 เอกสารที่ต้องใช้ในการเข้าระบบ

| รายการ | รายละเอียด |
|--------|-------------|
| แบบ ปสส.1-03 | แบบขึ้นทะเบียนผู้ประกันตน |
| สำเนาใบอนุญาตทำงาน | ต้องยังคง Valid |
| สำเนาหนังสือเดินทาง หรือ บัตรประจำตัวคนซึ่งไม่มีสัญชาติไทย | หนึ่งในสองอย่าง |
| ใบรับรองสุขภาพ | จากโรงพยาบาลรัฐ |

### 4.3 อัตราการหักประกันสังคม (อัพเดท พ.ศ. 2567)

| ประเภท | อัตราการหัก |
|---------|--------------|
| สังคมศึกษา อายุ | 5% ของเงินเดือน (สูงสุด 15,000 บาท) |
| สังคมสุขภาพ | 1.5% ของเงินเดือน (สูงสุด 5,000 บาท) |
| สังคมเล Danger | 0.5% ของเงินเดือน (สูงสุด 1,500 บาท) |

**หมายเหตุ**: แรงงานต่างชาติสามารถเลือกไม่หักประกันสังคมได้ หากมีประกันสังคมจากประเทศตนเอง

## 5. เอกสารที่ต้องมี

### 5.1 เอกสารแรงงานต่างชาติ (Update พ.ศ. 2569)

| เลขที่ | ชื่อเอกสาร | จำเป็นต่อการ? |
|--------|-------------|----------------|
| 1 | หนังสือเดินทาง (Passport) | จำเป็น |
| 2 | วีซ่าทำงาน (Work Visa) | จำเป็น |
| 3 | ใบอนุญาตทำงาน | จำเป็น |
| 4 | แผนกล่าวที่พัก (Notification of Residence) | จำเป็น |
| 5 | ใบรับรองสุขภาพ | จำเป็น |
| 6 | ประกันสุขภาพ | จำเป็น |

### 5.2 เอกสารพนักงานไทย

| เลขที่ | ชื่อเอกสาร | จำเป็นต่อการ? |
|--------|-------------|----------------|
| 1 | บัตรประชาชน | จำเป็น |
| 2 | บัตรประกันสังคม | จำเป็น |
| 3 | ใบเสร็จรับเงิน/สัญญาจ้าง | จำเป็น |

## 6. คำนวณสิทธิและเงินโบนัส

### 6.1 สูตรคำนวญเงินเดือน

```
เงินเดือนสุทธิ = เงินเดือนพื้นฐาน + เงินเพิ่ม + โบนัส - ภาษี - ประกันสังคม
```

### 6.2 สูตรคำนวญวันหยุดสะสม

```
วันหยุดสะสม = 8% × เงินเดือน / 30 × วันทำงาน
# ตัวอยน: เงินเดือน 15,000 บาท = 400 บาท/วัน
```

### 6.3 สูตรคำนวญภาษีเงินได้ (ภาษีหักณท์)

| เงินเดือนสุทธิ (บาท) | อัตราภาษี |
|----------------------|-----------|
| 0 - 150,000 | ยกเว้น |
| 150,001 - 300,000 | 5% |
| 300,001 - 500,000 | 10% |
| 500,001 - 750,000 | 15% |
| 750,001 - 1,000,000 | 20% |
| 1,000,001 - 2,000,000 | 25% |
| 2,000,001 - 5,000,000 | 30% |
| มากกว่า 5,000,000 | 35% |

## 7. คำแนะนำและข้อควรระวัง

### 7.1 ข้อควรระวัง

- **ควบคุมสัดส่วน**: หากเกินสัดส่วนที่กำหนด จะถูกลงโทษปรับ 5,000 - 100,000 บาท
- **เอกสารหมดอายุ**: ควรต่ออายุก่อนหมดอายุ 15 วัน
- **การหักประกันสังคม**: แรงงานต่างชาติสามารถเลือกไม่หักได้ แต่ควรแจ้งเป็นลายลักษภ์

- ระบบออนไลน์แรงงานต่างด้าว: https://alien.mol.go.th
- สำนักงานประกันสังคม: https://www.sso.go.th
