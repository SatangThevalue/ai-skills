# การบัญชีและการวางแผนการเงินส่วนบุคคลไทย (Thai Personal Accounting & Finance)

## Overview
สกิลนี้เป็นคู่มือปฏิบัติและเครื่องมือคำนวณสำหรับการทำบัญชีครัวเรือน/ส่วนบุคคล การจัดทำงบการเงินส่วนบุคคล (งบดุลและงบกระแสเงินสด) และการตรวจสุขภาพทางการเงินด้วยดัชนีอัตราส่วนการเงินส่วนบุคคล (Personal Financial Ratios) ตามแนวทาง Happy Money ของตลาดหลักทรัพย์แห่งประเทศไทย (SET) รวมถึงการบูรณาการความรู้ภาษีบุคคลธรรมดาเพื่อประโยชน์สูงสุดในการวางแผนทางการเงิน

## When to Use
- เมื่อต้องการเริ่มต้นหรือทบทวนระบบบันทึกรายรับ-รายจ่ายส่วนบุคคล (บัญชีครัวเรือน)
- เมื่อต้องการประเมินสุขภาพการเงินผ่านการคำนวณงบดุลและงบกระแสเงินสดประจำไตรมาส/ปี
- เมื่อต้องการตรวจเช็กอัตราส่วนการเงินส่วนบุคคล เช่น อัตราส่วนสภาพคล่อง, อัตราส่วนการออม, อัตราส่วนชำระหนี้สิน, อัตราส่วนความอยู่รอด และอัตราส่วนความมั่งคั่ง
- เมื่อต้องการคำนวณเป้าหมายเงินสำรองฉุกเฉินและการจัดสรรการออมเพื่อการลดหย่อนภาษี (SSF, RMF, Thai ESG)

## 1. โครงสร้างงบการเงินส่วนบุคคล (Personal Financial Statements)

เพื่อสะท้อนภาพรวมการเงินที่แท้จริง ต้องจัดทำงบการเงิน 2 งบหลัก ได้แก่:

### A. งบกระแสเงินสดส่วนบุคคล (Personal Cash Flow Statement)
แสดง **รายรับ** และ **รายจ่าย** ในรอบระยะเวลาหนึ่ง (เช่น ทุกเดือน) โดยใช้เกณฑ์เงินสด (Cash Basis):

| รายการ | รายละเอียด / ตัวอย่าง |
| :--- | :--- |
| **1. รายได้รวม (Total Income)** | |
| - รายได้จากการทำงาน (Active Income) | เงินเดือน, ค่าจ้างฟรีแลนซ์, โบนัส, รายได้จากเพจ Facebook |
| - รายได้จากสินทรัพย์ (Passive Income) | ดอกเบี้ยเงินฝาก, เงินปันผลหุ้น, ค่าเช่าอสังหาริมทรัพย์, กำไรจากระบบเทรด (ที่ถอนออกมา) |
| **2. รายจ่ายรวม (Total Expenses)** | |
| - เงินออมและลงทุน (Savings & Investments) | เงินออมฉุกเฉิน, กองทุนลดหย่อนภาษี (SSF/RMF/Thai ESG), พอร์ตลงทุนหุ้น/Crypto |
| - รายจ่ายคงที่ (Fixed Expenses) | ค่าผ่อนบ้าน, ค่าผ่อนรถ, เบี้ยประกันชีวิต/ประกันสุขภาพ, ค่าเช่าหอพัก |
| - รายจ่ายผันแปร (Variable Expenses) | ค่าอาหาร (เช่น ข้าวหน้าไก่ทอดกระเทียม), ค่าน้ำไฟ, ค่าเดินทาง, ช้อปปิ้ง, พักผ่อน |
| **กระแสเงินสดสุทธิ (Net Cash Flow)** | **รายได้รวม - รายจ่ายรวม (รวมเงินออมแล้ว)** |

*หมายเหตุ: กระแสเงินสดสุทธิควรเป็นบวกหรือเท่ากับศูนย์ (หากออมเงินครบแล้ว) หากติดลบแสดงว่าต้องดึงเงินสำรองมาใช้หรือก่อหนี้เพิ่ม*

### B. งบดุลส่วนบุคคล (Personal Balance Sheet)
แสดง **สถานะความมั่งคั่งสุทธิ** ณ เวลาใดเวลาหนึ่ง (เช่น ทุกสิ้นปีหรือสิ้นไตรมาส):

$$\text{ความมั่งคั่งสุทธิ (Net Worth)} = \text{สินทรัพย์รวม (Total Assets)} - \text{หนี้สินรวม (Total Liabilities)}$$

#### 1. สินทรัพย์ (Assets) แบ่งเป็น 3 ประเภท:
1. **สินทรัพย์สภาพคล่อง (Liquid Assets):** เงินสดในมือ, เงินฝากออมทรัพย์/ประจำ, กองทุนรวมตลาดเงิน (Money Market Fund)
2. **สินทรัพย์เพื่อการลงทุน (Investment Assets):** หุ้นสามัญ, กองทุนรวมทั่วไป, สินทรัพย์ดิจิทัล (Crypto), ประกันชีวิตแบบสะสมทรัพย์ (ใช้มูลค่าเวนคืนกรมธรรม์), พอร์ตลงทุนส่วนตัว
3. **สินทรัพย์ส่วนตัว (Personal Assets):** บ้านที่อยู่อาศัย, รถยนต์ใช้งาน, ของสะสมมีมูลค่า, สินค้าไอที/เครื่องมือทำงาน

#### 2. หนี้สิน (Liabilities) แบ่งเป็น 2 ประเภท:
1. **หนี้สินระยะสั้น (Short-term Liabilities):** หนี้บัตรเครดิต, หนี้สินส่วนบุคคล, หนี้เงินยืมระยะสั้น (ชำระคืนภายใน 1 ปี)
2. **หนี้สินระยะยาว (Long-term Liabilities):** หนี้เพื่อที่อยู่อาศัย (สินเชื่อบ้าน), หนี้เช่าซื้อรถยนต์ (ส่วนที่ยังผ่อนไม่หมด)

## 2. อัตราส่วนทางการเงินส่วนบุคคลที่สำคัญ (Personal Financial Ratios)
อ้างอิงจากเกณฑ์การประเมินสุขภาพการเงินของตลาดหลักทรัพย์แห่งประเทศไทย (SET):

| อัตราส่วนทางการเงิน | สูตรคำนวณ | ค่ามาตรฐาน / เกณฑ์ผ่าน | คำอธิบายความหมาย |
| :--- | :--- | :--- | :--- |
| **1. อัตราส่วนสภาพคล่องพื้นฐาน** (Basic Liquidity Ratio) | $$\frac{\text{สินทรัพย์สภาพคล่อง}}{\text{ค่าใช้จ่ายรวมต่อเดือน}}$$ | **3 - 6 เท่า** | บ่งบอกว่าถ้าขาดรายได้กระทันหัน จะมีเงินใช้จ่ายได้อีกกี่เดือน (ถ้าเป็นฟรีแลนซ์/นักเทรดควรมี 6-12 เท่า) |
| **2. อัตราส่วนการออม** (Saving Ratio) | $$\frac{\text{เงินออมและลงทุนต่อเดือน}}{\text{รายได้รวมต่อเดือน}} \times 100$$ | **> 10%** ขึ้นไป | ความสามารถในการสะสมความมั่งคั่งเพื่อเป้าหมายในอนาคต (เป้าหมายที่ดีคือ 20-30%) |
| **3. อัตราส่วนการชำระคืนหนี้สิน** (Debt Service Ratio - DSR) | $$\frac{\text{ยอดผ่อนชำระหนี้ต่อเดือน}}{\text{รายได้รวมต่อเดือน}} \times 100$$ | **ไม่ควรเกิน 35% - 45%** | ความสามารถในการรับภาระหนี้ หากสูงเกินไปจะเสี่ยงต่อปัญหาฝืดเคืองทางการเงิน |
| **4. อัตราส่วนหนี้สินต่อสินทรัพย์** (Debt to Asset Ratio) | $$\frac{\text{หนี้สินรวม}}{\text{สินทรัพย์รวม}} \times 100$$ | **ไม่ควรเกิน 50%** | สัดส่วนหนี้สินเทียบกับทรัพย์สินทั้งหมดที่มี หากเกิน 50% แสดงว่าฐานะเริ่มไม่มั่นคง |
| **5. อัตราส่วนความอยู่รอด** (Survival Ratio) | $$\frac{\text{รายได้จากการทำงาน + รายได้จากสินทรัพย์}}{\text{ค่าใช้จ่ายรวมต่อเดือน}}$$ | **> 1 เท่า** | บ่งบอกว่ารายได้ทั้งหมดเพียงพอกับรายจ่ายหรือไม่ (ต้องมากกว่า 1 เท่าเพื่อให้อยู่รอดได้) |
| **6. อัตราส่วนความมั่งคั่ง** (Wealth Ratio) | $$\frac{\text{รายได้จากสินทรัพย์ (Passive Income)}}{\text{ค่าใช้จ่ายรวมต่อเดือน}}$$ | **>= 1 เท่า** | ตัวชี้วัดระดับ "อิสรภาพทางการเงิน" หากเท่ากับหรือมากกว่า 1 แสดงว่าสามารถอยู่ได้โดยไม่ต้องพึ่งพารายได้จากการทำงาน |

## 3. ขั้นตอนการลงบันทึกและการประเมินผลทางการเงิน (SOP)

1. **จดบันทึกรายรับ-รายจ่ายรายวัน:**
   - ใช้แอปพลิเคชัน (เช่น Happy Money App) หรือตาราง Excel/Google Sheets
   - แยกประเภทรายการให้ชัดเจนเพื่อประเมินค่าใช้จ่ายที่จำเป็นและไม่จำเป็น
2. **สรุปงบกระแสเงินสดทุกสิ้นเดือน:**
   - นำตัวเลขรวมรายรับและรายจ่ายมาลงตาราง
   - คำนวณกระแสเงินสดสุทธิ และแยกยอดเงินที่ถูกจัดสรรไปออม/ลงทุน
3. **จัดทำงบดุลส่วนบุคคลทุกสิ้นไตรมาส/ปี:**
   - อัปเดตมูลค่าตลาดของสินทรัพย์ (เช่น ยอดเงินในบัญชี, มูลค่าพอร์ตหุ้น/Crypto ล่าสุด)
   - อัปเดตยอดหนี้คงเหลือจริงจากใบแจ้งหนี้
   - คำนวณความมั่งคั่งสุทธิ (Net Worth) เพื่อดูทิศทางการเติบโตของทรัพย์สิน
4. **คำนวณอัตราส่วนทางการเงินเพื่อประเมินสุขภาพ (Financial Checkup):**
   - รันสคริปต์ตรวจสุขภาพการเงินประจำงวดเพื่อประเมินผลตามเกณฑ์ SET
   - วางแผนปรับปรุงส่วนที่มีสัญญาณเตือนภัย (เช่น หาก DSR สูงเกินไป ต้องงดก่อหนี้เพิ่มและเน้นโปะหนี้ดอกเบี้ยสูง)
5. **วางแผนภาษีช่วงปลายปี (ไตรมาส 4):**
   - คำนวณประมาณการรายได้ทั้งปี หักค่าใช้จ่ายและค่าลดหย่อนพื้นฐาน (อ้างอิง `thai-finance-legal-guide`)
   - คำนวณสิทธิ์ซื้อกองทุนเพื่อลดหย่อนภาษี เช่น SSF (สูงสุด 30% ของรายได้ ไม่เกิน 200,000 บาท), RMF (สูงสุด 30% ไม่เกิน 500,000 บาท), Thai ESG (สูงสุด 30% ไม่เกิน 300,000 บาท)
   - ดำเนินการซื้อให้เสร็จสิ้นภายในวันทำการสุดท้ายของปีภาษีนั้นๆ

## Common Pitfalls
1. **ประเมินมูลค่าสินทรัพย์ส่วนตัวสูงเกินไป:** เช่น มูลค่ารถยนต์ใช้งานที่ตกลงทุกปี หรือสิ่งของสะสมที่สภาพคล่องต่ำ ควรคำนวณสินทรัพย์ด้วยราคาที่สามารถขายได้จริง ณ ปัจจุบัน (Market Value) หรือไม่นำมารวมในการคำนวณแผนเกษียณ
2. **ละเลยค่าใช้จ่ายรายปีเฉลี่ยเป็นรายเดือน:** เช่น ค่าเบี้ยประกัน, ภาษีสังคม, ค่าบำรุงรักษารถ ที่จ่ายเป็นรายปี ทำให้กระแสเงินสดเดือนนั้นๆ ติดลบรุนแรง ควรเฉลี่ยค่าใช้จ่ายเหล่านี้เป็นรายเดือนและกันเงินสะสมไว้ล่วงหน้า
3. **สับสนระหว่าง SSF และ RMF:** SSF เน้นถือครอง 10 ปีปฏิทิน ไม่มีขั้นต่ำ ส่วน RMF ต้องถือครองอย่างน้อย 5 ปี และต้องมีอายุครบ 55 ปีบริบูรณ์ พร้อมเงื่อนไขต้องซื้อต่อเนื่องทุกปี (หรือเว้นได้ไม่เกิน 1 ปีติดต่อกัน) การผิดเงื่อนไขจะโดนเรียกคืนภาษีย้อนหลัง
4. **ไม่แยกพอร์ตลงทุนความเสี่ยงสูง (Satellite Portfolio):** เช่น พอร์ตระบบเทรดอัจฉริยะ (Forex/Crypto) หรือการเก็งกำไร ควรคุมสัดส่วนไม่ให้เกิน 30% ของสินทรัพย์ลงทุนทั้งหมด และห้ามนำเงินสำรองฉุกเฉินมาใส่ในพอร์ตเก็งกำไรนี้เด็ดขาด

## Verification Checklist
- [ ] มีบัญชีเงินสำรองฉุกเฉินแยกต่างหาก และมีสภาพคล่องสูงเพียงพอสำหรับค่าใช้จ่าย 3-6 เดือนขึ้นไป
- [ ] จัดทำงบดุลและงบกระแสเงินสดอัปเดตล่าสุดเรียบร้อยแล้ว
- [ ] อัตราส่วน DSR (ภาระหนี้) ไม่เกิน 45% ของรายได้รวม
- [ ] มีการคำนวณและตรวจสอบสิทธิ์ลดหย่อนภาษีก่อนตัดสินใจซื้อกองทุน SSF / RMF / Thai ESG
- [ ] ตรวจสอบว่าไม่เกิดการซื้อกองทุนลดหย่อนภาษีเกินสิทธิ์กฎหมายกำหนด (Retirement cap รวม 500k, Thai ESG แยก 300k)
