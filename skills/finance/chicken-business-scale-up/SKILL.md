# ยกระดับธุรกิจไก่สด — จากร้านในตลาดสู่ระดับภาคและประเทศ

Skill นี้รวบรวมกลยุทธ์ ขั้นตอน และข้อควรระวังในการขยายธุรกิจขายไก่สดจากระดับร้านค้าในตลาด ไปสู่ระดับภูมิภาคหรือระดับประเทศ อิงจากงานวิจัย KKU (SWOT ร้านลำไยไก่สด, 2554), KResearch (2568), Krungsri Industry Outlook (2569–2571) และ SCB EIC

ไม่ครอบคลุมการส่งออกต่างประเทศ (ต้องใช้ใบอนุญาต GMP/HACCP ระดับโรงงาน) หรือการเลี้ยงไก่ในฟาร์ม

---

## When to Use

- วางแผนขยายร้านขายไก่จากตลาดท้องถิ่นไปหลายจังหวัด
- ต้องการวิเคราะห์จุดแข็ง/จุดอ่อนของธุรกิจไก่สดเพื่อ Scale Up
- ออกแบบระบบ Supply Chain และ Cold Chain สำหรับธุรกิจไก่
- ประเมินความเป็นไปได้ในการแปรรูปหรือขายส่ง

---

## ภาพรวมอุตสาหกรรมไก่ไทย (อ้างอิง: Krungsri/KResearch 2568-2569)

| ตัวชี้วัด | ข้อมูล |
|----------|--------|
| ผู้ประกอบการในอุตสาหกรรม | 1,278 ราย (เฉพาะนิติบุคคล) |
| รายย่อย (Micro/Small) | ~80% ของผู้เล่นทั้งหมด แต่ผลิตแค่ 10% ของปริมาณ |
| รายใหญ่ | ~20% แต่ควบคุม 90% ของผลผลิต |
| Gross Profit Margin | **20–30%** |
| แหล่งผลิตหลัก | ภาคกลาง 73.7% → อีสาน 11.9% → เหนือ 8.0% → ใต้ 6.4% |
| ราคาหน้าฟาร์ม (2568) | ~42.0 บาท/กก. |
| การบริโภคในประเทศ | 70% ของผลผลิตทั้งหมด |

> **โอกาส:** ไก่เป็นโปรตีนราคาถูกสุด รองรับทุกศาสนา เติบโตตามกระแสสุขภาพ และรายย่อยยังมีพื้นที่เหลือในตลาดท้องถิ่น-ภูมิภาค

## Procedure

### Phase 1 — วิเคราะห์สถานะปัจจุบัน (SWOT)

ก่อน Scale Up ต้องรู้ว่าตัวเองอยู่ที่ไหน ใช้กรอบ SWOT เฉพาะธุรกิจไก่สด:

#### จุดแข็งที่ต้องมีก่อน Scale Up
- [ ] ฐานลูกค้าประจำในตลาดปัจจุบัน ≥ 60% ของยอดขาย
- [ ] ยอดขาย/วันทรงตัวหรือเติบโต ≥ 3 ปีต่อเนื่อง
- [ ] ระบบชำแหละที่สะอาด + ความสด = ชื่อเสียงในพื้นที่
- [ ] เครือข่ายซัพพลายเออร์ (ฟาร์ม/โรงฆ่า) ที่ไว้ใจได้

#### จุดอ่อนที่ต้องแก้ก่อนขยาย (จากวิจัย KKU)
- หน้าร้านและความสะอาดไม่ดึงดูด → ลงทุนปรับก่อน
- ไม่มีการทำโฆษณา พึ่งปากต่อปากอย่างเดียว → เริ่ม LINE OA / Facebook
- ไม่มีระบบบัญชีและต้นทุน → ต้องทำก่อนขยาย
- ขาดการแบ่งหน้าที่ชัดเจน → วางโครงสร้างองค์กรก่อน Scale

---

### Phase 2 — เลือกรูปแบบการ Scale Up

| รูปแบบ | เหมาะกับ | ต้นทุนเริ่มต้น | ความเสี่ยง |
|--------|---------|--------------|----------|
| **เปิดสาขาในตลาดอื่น** | มีทุนน้อย ต้องการควบคุมคุณภาพ | ต่ำ | ต่ำ |
| **ขายส่งให้ร้านอาหาร/โรงแรม** | มีกำลังผลิตเหลือ | ต่ำมาก | กลาง (ชำระช้า) |
| **ซัพพลายให้ตลาดสด/ห้างฯ** | มีใบอนุญาตและมาตรฐาน | กลาง | กลาง |
| **แปรรูปขั้นต้น (ชิ้นส่วน/มารินเนด)** | มีพื้นที่+เครื่องมือ | กลาง-สูง | กลาง |
| **Franchise/ระบบแฟรนไชส์** | มีแบรนด์แข็งแกร่ง | สูง | ต่ำ (โอนความเสี่ยงให้แฟรนไชซี) |

### Phase 3 — ขั้นตอนการ Scale Up สู่ระดับภาค (5 ขั้น)

#### ขั้นที่ 1: กำหนด Capacity และ Break-even ให้ชัด

```
ตัวอย่าง:
- ต้นทุนไก่มีชีวิต: 40 บาท/กก.
- น้ำหนักไก่มีชีวิต → เนื้อไก่: ~70% (ทิ้งขนและเครื่องใน)
- ต้นทุนต่อกก.เนื้อ: 40/0.7 = ~57 บาท/กก.
- ราคาขายปลีก: 70-80 บาท/กก. → Margin ~20-30%
- ต้องขาย X กก./วัน เพื่อคุ้มต้นทุนการขยาย
```

ใช้ `execute_code` คำนวณ Break-even ก่อนเสมอ

#### ขั้นที่ 2: สร้างระบบ Cold Chain

| องค์ประกอบ | รายละเอียด | ข้อควรระวัง |
|-----------|-----------|------------|
| **รถขนส่งแบบแช่เย็น** | ต้องควบคุม 0–4°C ตลอดเส้นทาง | เช่าก่อนซื้อถ้างบจำกัด |
| **ห้องเย็น (Cold Room)** | ขั้นต่ำ 4–10°C สำหรับไก่สด ≤24 ชม. | ไฟฟ้าเป็นต้นทุนสำคัญ |
| **เวลาชำแหละ** | ควรก่อนตี 3 เพื่อส่งถึงตลาดตอนเช้า | ต้องบริหารแรงงานกะดึก |
| **อุณหภูมิ** | ไก่สด: 0–4°C / ไก่แช่แข็ง: -18°C | ทุก 4°C ที่ขึ้น = อายุสั้นลงครึ่ง |

#### ขั้นที่ 3: สร้างระบบจัดซื้อ (Upstream)

- **Contract Farming:** ตกลงราคาล่วงหน้ากับฟาร์มเพื่อล็อกต้นทุน
- **Multiple Suppliers:** มีซัพพลายเออร์ ≥ 2 รายเพื่อลดความเสี่ยง
- **ต้นทุนอาหารสัตว์:** คิดเป็น 60–70% ของต้นทุนการเลี้ยง — ติดตามราคาข้าวโพด/กากถั่วเหลือง
- ปี 2569: ไทยเปิดนำเข้าข้าวโพดจากสหรัฐฯ **ถูกกว่าแหล่งเดิม 47–118 USD/ตัน** = โอกาสต้นทุนลด

#### ขั้นที่ 4: ขยายช่องทางจำหน่าย (Downstream)

**ลำดับการขยายที่แนะนำ:**
```
ตลาดสดเดิม (Base)
  → ร้านอาหาร/ครัวโรงแรมในจังหวัด (B2B ปริมาณมาก)
    → ตลาดสดอำเภอใกล้เคียง (Clone Model)
      → ซัพพลาย Modern Trade / ห้างฯ ภูมิภาค
        → แปรรูปขั้นต้น (ชิ้นส่วน, มารินเนด)
          → ระดับภาค (กระจายหลายจังหวัด)
```

**ลูกค้า B2B ที่ควรเจาะก่อน:**
| กลุ่ม | ปริมาณ/สัปดาห์ | ข้อดี | ข้อควรระวัง |
|------|--------------|------|------------|
| ร้านอาหาร/ภัตตาคาร | 50–500 กก. | สม่ำเสมอ ต้องการคุณภาพ | เครดิต 7–30 วัน |
| โรงเรียน/มหาวิทยาลัย | 100–1,000 กก./อาทิตย์ | ปริมาณมาก สม่ำเสมอ | ประมูล ราคากดดัน |
| โรงแรม 3–5 ดาว | 50–300 กก. | ราคาดีกว่า ต้องการคุณภาพสูง | ต้องผ่านมาตรฐาน |
| ร้านสะดวกซื้อ (Franchise) | ตามสาขา | ปริมาณแน่นอน | เงื่อนไขซัพพลายเออร์เข้มงวด |

#### ขั้นที่ 5: สร้างแบรนด์และมาตรฐาน

- **ตราสินค้า:** ชื่อ-โลโก้-บรรจุภัณฑ์ที่จดจำได้ (แม้แค่ถุง+สติ๊กเกอร์)
- **ใบอนุญาต:** ใบอนุญาตโรงฆ่าสัตว์ + สุขาภิบาลอาหาร (สอบถาม กรมปศุสัตว์)
- **มาตรฐาน GMP เบื้องต้น:** สะอาด มีการบันทึก Lot ผลิต
- **ระบบ QC:** ตรวจสีเนื้อ กลิ่น อุณหภูมิทุก Lot ก่อนส่ง

---

### Phase 4 — ขยายสู่ระดับประเทศ

เมื่อ Scale ภาคได้แล้ว จึงเดินหน้าระดับประเทศ:

#### โมเดลที่ใช้ได้จริง

| โมเดล | ตัวอย่างบริษัทไทย | เหมาะเมื่อ |
|-------|----------------|----------|
| **Central Processing + Regional Distribution** | CPF, CFARM | มีทุนสร้างโรงงานกลาง |
| **Hub & Spoke** | โรงฆ่ากลาง 1 แห่ง + Depot ย่อยแต่ละภาค | กำลังผลิต >5,000 ตัว/วัน |
| **Franchise Network** | Franchise ร้านขายไก่ในตลาด | มีแบรนด์ + Know-how ที่ถ่ายทอดได้ |
| **OEM ให้ Retailer** | ผลิตให้ห้างฯ ใช้แบรนด์ตัวเอง | ไม่ต้องการแบรนด์ตัวเอง |

#### ปัจจัยที่ต้องมีก่อนขยายระดับประเทศ

- [ ] กำลังผลิต >3,000 ตัว/วัน
- [ ] ระบบบัญชีและ ERP (ติดตามต้นทุน-กำไรแต่ละสาขา)
- [ ] ทีมบริหาร: ผู้จัดการฝ่ายผลิต / โลจิสติกส์ / ขาย แยกกัน
- [ ] เงินทุนหมุนเวียน ≥ 3 เดือนของต้นทุน
- [ ] ใบอนุญาตโรงฆ่าสัตว์ระดับจังหวัดหรือ GMP

## ข้อแนะนำเฉพาะธุรกิจไก่

1. **ไก่เป็นสินค้าเน่าเสียง่าย** — ทุกการขยายต้องมี Cold Chain มาก่อน ห้ามข้ามขั้นตอน
2. **ไก่เป็นสินค้าควบคุมราคา** — ราคาขยับขึ้นได้จำกัด ต้องบริหารต้นทุนอย่างเคร่งครัด
3. **ปัจจัยสำคัญต่อพฤติกรรมซื้อ (วิจัย KKU):** ความสะดวก > ราคา > บริการ > คุณภาพ > โปรโมชัน
4. **ลูกค้าหลักคือผู้หญิง อายุ 40–50 ปี** — Communication ต้องเข้าถึงกลุ่มนี้ (LINE > Facebook)
5. **ชั่วโมงขายคือ 04.00–13.00** — ต้องบริหารกำลังผลิตและโลจิสติกส์ให้เสร็จก่อนเที่ยง

---

## ข้อควรระวัง (Pitfalls)

- **อย่าขยายก่อนมีระบบ Cold Chain** — ไก่เน่า = ลูกค้าหายถาวร
- **อย่าพึ่งลูกค้ารายเดียว >30% ของยอดขาย** — หากเขาเปลี่ยนซัพพลายเออร์จะกระทบรุนแรง
- **ระวังเครดิต B2B** — ร้านอาหารบางรายขอเครดิต 30–60 วัน กระทบ Cash Flow
- **รายใหญ่ครอง 90% ของตลาด** — อย่าแข่งในเกมราคา ต้องหา Niche (ไก่ออร์แกนิก/ไก่พื้นเมือง/บริการส่งตรง)
- **ต้นทุนอาหารสัตว์คิด 60–70% ของต้นทุนเลี้ยง** — หากซื้อจากฟาร์มภายนอก ต้องติดตามราคาข้าวโพด/ถั่วเหลืองทุกเดือน
- **กฎหมายที่ต้องรู้:** พ.ร.บ.ควบคุมการฆ่าสัตว์ พ.ศ.2559, ใบอนุญาต อ.4 (โรงฆ่า), ใบอนุญาตสุขาภิบาลอาหาร

## Verification

Skill นี้ทำงานถูกต้องเมื่อสามารถ:
- วิเคราะห์ SWOT ของร้านไก่สดและระบุจุดที่ต้องแก้ก่อน Scale
- คำนวณ Break-even และ Capacity ที่จำเป็นก่อนขยาย
- เลือกรูปแบบการ Scale ที่เหมาะกับทุน/กำลังผลิตปัจจุบัน
