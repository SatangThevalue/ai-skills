# Social SEO Checklist 2026 — 10 ขั้นตอน

ใช้เมื่อ: วางแผน content strategy, ทำ Social Media Marketing, ปรับ profile ให้ค้นเจอ,
หรือ audit performance ของช่องทาง social ทุกประเภท

---

## ✅ ข้อควรทำ (DOs)

- ทำ keyword research **ก่อน** เริ่มเขียน content ทุกครั้ง
- ใส่ keyword หลักในตำแหน่ง **แรกสุด** ของ caption/bio
- อัปเดต performance audit **ทุกเดือน** และปรับ strategy ตาม data
- ทดสอบ content รูปแบบต่างกัน (A/B test) อย่างน้อยเดือนละ 1 ครั้ง
- ปรับ format ให้เหมาะกับแต่ละ platform — ห้ามใช้ content เดิมทุก platform

## ⚠️ ข้อควรระวัง (DON'Ts)

- **อย่า** ใช้ hashtag เกิน 10 ตัวใน Instagram (algorithm ลดการ reach)
- **อย่า** โพสต์แบบ "ขาย" มากกว่า 30% ของ content ทั้งหมด
- **หลีกเลี่ยง** keyword stuffing — ใส่ keyword มากเกินทำให้อ่านไม่รู้เรื่อง
- **อย่า** ละเลย comment ใน 2 ชั่วโมงแรก — ช่วง peak engagement สำคัญมาก
- **ห้าม** copy content ข้าม platform โดยไม่ปรับ format เลย

## ขั้นตอนที่ 1: Keyword Research (Long-tail Keywords)

### เครื่องมือที่ใช้

| เครื่องมือ | Platform | ประโยชน์ |
|-----------|----------|---------|
| Google Keyword Planner | Google | Search volume + competition |
| TikTok Creative Center | TikTok | Trending keywords in-app |
| Pinterest Trends | Pinterest | Visual content trends |
| Instagram Search | Instagram | Hashtag popularity |
| Answer the Public | ทุก platform | คำถามที่คนค้นหา |

### เป้าหมาย Keyword

- **Long-tail keywords:** 3-5 คำ เช่น "ซื้อหุ้นไทยสำหรับมือใหม่" แทน "หุ้นไทย"
- Search volume target: **100-1,000/เดือน** (แข่งได้จริง)
- ใช้ 1 primary keyword + 2-3 secondary keywords ต่อโพสต์
- ตรวจ competitor ว่าเขาใช้ keyword อะไร (ค้นใน platform ดู top posts)

### ขั้นตอนปฏิบัติ

```
1. เปิด TikTok Creative Center → ค้น topic หลัก
2. จด keywords ที่ trending 7 วันล่าสุด
3. Cross-check ใน Google Keyword Planner
4. เลือก 5-10 long-tail keywords สำหรับเดือนนี้
5. บันทึกลง keyword list ของ knowledge bank
```

## ขั้นตอนที่ 2: Caption Optimization

### โครงสร้าง Caption ที่ดี

```
[Keyword หลัก + Hook] — ใน 10 ตัวอักษรแรก
[ขยายความ / ปัญหาที่แก้ได้]
[Social Proof / ตัวเลข / ข้อมูล]
[Call-to-Action ธรรมชาติ]
[Hashtags]
```

### ตัวอย่าง Caption ที่ดี vs ไม่ดี

**❌ ไม่ดี:**
> "วันนี้เราอยากมาแชร์เรื่องการลงทุนนะครับ มันเป็นเรื่องที่สำคัญมากและทุกคนควรรู้..."

**✅ ดี:**
> "ลงทุนหุ้นไทยด้วย 1,000 บาทแรก — นี่คือสิ่งที่ผมอยากรู้ตั้งแต่ 5 ปีที่แล้ว
> ตอนนี้ port เติบโตแล้ว 340% ด้วยหลักการง่ายๆ 3 ข้อ
> คอมเมนท์ 'ส่ง' รับ checklist ฟรีครับ 👇
> #ลงทุนหุ้นไทย #หุ้นไทยมือใหม่ #การเงิน"

### Caption Length แนะนำแต่ละ Platform

| Platform | ความยาวแนะนำ | หมายเหตุ |
|----------|------------|---------|
| Instagram | 150-300 ตัวอักษร | Hook สำคัญ — แสดงแค่ 125 ตัวแรก |
| TikTok | 100-150 ตัวอักษร | ใส่ keyword ให้ครบก่อน hashtag |
| Facebook | 200-400 ตัวอักษร | Storytelling ได้ยาวกว่า |
| LinkedIn | 300-600 ตัวอักษร | Professional tone |
| X (Twitter) | 200-280 ตัวอักษร | Punchy, direct |

## ขั้นตอนที่ 3: Alt Text for Images

### วิธีเขียน Alt Text ที่ดี

```
[สิ่งที่เห็นในภาพ] + [keyword] + [context ที่เกี่ยวข้อง]
```

**ตัวอย่าง:**
- ❌ "IMG_2034.jpg"
- ❌ "ภาพสวย"
- ✅ "นักลงทุนรายย่อยตรวจ port หุ้นไทยผ่านแอป SETTRADE บนสมาร์ทโฟน"

### Platform ที่รองรับ Alt Text

| Platform | วิธีเพิ่ม Alt Text |
|----------|-----------------|
| Instagram | Edit Photo → Accessibility → Write Alt Text |
| Facebook | เปิดโพสต์ → Edit Alt Text |
| Twitter/X | Add Description ตอน upload |
| Pinterest | Description field = Alt Text |

## ขั้นตอนที่ 4: Profile Optimization

### Checklist Profile ที่สมบูรณ์

- [ ] **Username:** ค้นหาง่าย ตรงกับ brand name
- [ ] **Profile Photo:** ชัด, หน้าตรง (สำหรับ personal brand)
- [ ] **Bio Line 1:** บอกว่าทำอะไร ใส่ keyword หลัก
- [ ] **Bio Line 2:** บอก target audience ("สำหรับ...")
- [ ] **Bio Line 3:** CTA ชัดเจน + ลิงก์
- [ ] **Link:** ใช้ Linktree/Beacons หากมีหลาย destination
- [ ] **Highlights (IG):** จัดหมวด Products, Reviews, FAQ, About
- [ ] **Category (FB/IG Business):** ตั้งให้ตรงกับธุรกิจ

### Template Bio

```
📊 [สิ่งที่ทำ] + [Keyword หลัก]
👥 ช่วย [target audience] ให้ [ผลลัพธ์ที่ได้]
👇 [CTA] → [ลิงก์]
```

**ตัวอย่าง:**
```
📊 Data Analyst สอนลงทุนหุ้นไทย
👥 ช่วยมือใหม่สร้าง passive income ด้วย data
👇 รับ Free Checklist → link.satang.app

## ขั้นตอนที่ 5: Content Sequence Design (Awareness → Decision)

### Customer Journey Framework

```
Stage 1: AWARENESS (รู้จักเรา)
→ โพสต์แนะนำตัว, behind the scenes, ค่านิยม
→ เป้า: reach กว้าง, ไม่ขาย

Stage 2: INTEREST (สนใจเรา)
→ Educational content, tips, how-to
→ เป้า: saves, shares, follows

Stage 3: CONSIDERATION (กำลังคิด)
→ Case studies, testimonials, comparisons
→ เป้า: comments, DM, link clicks

Stage 4: INTENT (เกือบจะซื้อ)
→ FAQ, pricing, what you get
→ เป้า: DM, inquiry

Stage 5: DECISION (พร้อมซื้อ)
→ Limited time offers, final CTA
→ เป้า: conversion
```

### สัดส่วน Content แนะนำต่อสัปดาห์

| ประเภท | สัดส่วน | วัตถุประสงค์ |
|--------|--------|------------|
| Educational | 40% | Awareness + Interest |
| Storytelling | 30% | Trust building |
| Social Proof | 20% | Consideration |
| Promotional | 10% | Conversion |

## ขั้นตอนที่ 6: Platform-Specific Adaptation

### Facebook
- ความยาว: 200-400 คำ (storytelling ได้)
- รูปแบบที่ดี: personal story → lesson → CTA
- เวลาโพสต์: วันธรรมดา 9:00-11:00 หรือ 19:00-21:00
- อย่า: ใช้ภาษาทางการเกินไป

### Instagram
- Hook ใน 2 บรรทัดแรก (แสดงก่อน "...more")
- Carousel: ให้ข้อมูล step-by-step ได้ดี
- Reels: 15-30 วินาทีสำหรับ reach; 60-90 วินาทีสำหรับ depth
- อย่า: caption ยาวโดยไม่มี line breaks

### TikTok
- Hook **5 วินาทีแรก** คือทุกอย่าง
- ใส่ text overlay สำคัญบนหน้าจอ (คนดูโดยไม่มีเสียง)
- ใช้ trending sounds ที่เกี่ยวข้อง
- อย่า: พูดช้าใน 5 วินาทีแรก

### LINE OA
- tone เป็นมิตร เหมือนคุยกับเพื่อน
- ส่ง broadcast ไม่เกิน 3 ครั้ง/สัปดาห์
- ใช้ Rich Message + Quick Reply สำหรับ engagement

### LinkedIn (ถ้าใช้)
- ขึ้นต้นด้วย hook ที่ provoking
- ใช้ line breaks บ่อย (อ่านง่ายบนมือถือ)
- Professional insights > promotional content

## ขั้นตอนที่ 7: Community Building

### Engagement Rules

| กิจกรรม | เวลาที่ควรทำ | เป้าหมาย |
|---------|------------|---------|
| ตอบ comment | ภายใน 2 ชั่วโมงแรก | Algorithm boost |
| ตอบ DM | ภายใน 4-24 ชั่วโมง | Relationship |
| Comment ใน post คนอื่น | 30 นาที/วัน | Visibility |
| Reshare UGC | 20% ของ content | Social proof |

### Community Building Tactics

1. **Pinned Comment:** ถามคำถามเพื่อกระตุ้นการตอบ
2. **Weekly Series:** เช่น "Monday Tips", "Friday Recap"
3. **Q&A Live:** เดือนละ 1 ครั้ง
4. **User-Generated Content (UGC):** ขอให้ลูกค้าแชร์ประสบการณ์
5. **Community Groups:** สร้าง Facebook Group หรือ LINE Group VIP

## ขั้นตอนที่ 8: Video Optimization

### Short-form Video (TikTok/Reels)

```
[0-5 วิ] Hook — บอกว่าดูแล้วได้อะไร
[5-15 วิ] Problem — ปัญหาที่คนเจอ
[15-40 วิ] Solution — วิธีแก้ชัดเจน
[40-55 วิ] Proof — ตัวอย่าง/ผลลัพธ์
[55-60 วิ] CTA — บอกให้ทำอะไรต่อ
```

### Long-form Video (YouTube)

- Thumbnail: หน้าชัด + text overlay + สีสดใส
- Title: keyword หลัก + benefit ชัดเจน (< 60 ตัวอักษร)
- Description: keyword ใน 2 บรรทัดแรก + timestamps + links
- ใส่ Chapter Timestamps ทุกครั้ง (ช่วย SEO ใน Google)

### Video Checklist

- [ ] Hook แรง ใน 5 วินาที
- [ ] Text overlay สำคัญ (สำหรับ silent viewers)
- [ ] Captions/Subtitles เปิดไว้เสมอ
- [ ] Thumbnail ดึงดูดใน thumbnail size เล็ก
- [ ] CTA ชัดเจนตอนท้าย

## ขั้นตอนที่ 9: Social Commerce Integration

### Platform Commerce Tools

| Platform | เครื่องมือ | วิธีใช้ |
|---------|----------|--------|
| Instagram | Shopping Tags | tag สินค้าในทุกโพสต์/Reel |
| TikTok | TikTok Shop | สร้าง product listing + video shopping |
| Facebook | Marketplace + Shop | catalog สินค้า + Live Shopping |
| LINE | LINE Shopping | เชื่อม LINE OA กับ Shopping |

### Social Commerce Best Practices

1. ใส่ product link ใน bio และ stories เสมอ
2. ใช้ Live Shopping สำหรับ new launch (conversion rate สูงกว่า 3-5x)
3. ทำ product review video ก่อน link ซื้อ
4. ใส่ price ชัดเจนในโพสต์ (ลด friction)
5. เพิ่ม scarcity/urgency จริง: "เหลือ 20 ชิ้น" (ไม่ใช่ fake scarcity)

## ขั้นตอนที่ 10: Performance Audit

### Metrics ที่ต้องตรวจทุกเดือน

| Metric | เป้าหมาย | แหล่งข้อมูล |
|--------|---------|------------|
| Engagement Rate | 3-6% | Platform Analytics |
| Reach Growth | +10%/เดือน | Platform Analytics |
| Follower Growth | +5-15%/เดือน | Platform Analytics |
| Link Clicks | ขึ้นอยู่กับ funnel | Linktree / UTM |
| Profile Visits | ตามเป้า traffic | Platform Analytics |
| Top Posts | 3 โพสต์ที่ดีที่สุด | Platform Analytics |

### Monthly Audit Process

```
สัปดาห์ที่ 4 ของเดือน:
1. Export data จาก platform analytics (7 วัน / 30 วัน)
2. คำนวณ average engagement rate
3. ดู top 3 posts — มี pattern อะไร?
4. ดู worst 3 posts — ปัญหาคืออะไร?
5. ตรวจ keyword ranking (ค้นหาใน platform ดูว่าเราอยู่อันดับไหน)
6. อัปเดต knowledge bank ด้วย insights ใหม่
7. วางแผน content เดือนหน้าจาก data นี้
```

### การคำนวณ Engagement Rate

```
Engagement Rate = (Likes + Comments + Shares + Saves) / Reach × 100

เป้าหมาย:
- < 1% = ต้องปรับ content strategy ด่วน
- 1-3% = ปานกลาง ปรับ caption และ posting time
- 3-6% = ดี ทำต่อ
- > 6% = ยอดเยี่ยม ศึกษา pattern แล้ว scale
```

```

- Location hashtag: 1-2 ตัว (ถ้า local business)

## Pitfalls ที่ควรหลีกเลี่ยง

- อย่า keyword stuff — ใส่ keyword ซ้ำๆ มากเกินทำให้อ่านไม่รู้เรื่อง
- อย่า copy content ข้าม platform โดยไม่ปรับ format
- หลีกเลี่ยง hashtag ที่ banned (ตรวจก่อนใช้)
- อย่า audit แค่ vanity metrics (likes) โดยไม่ดู business metrics (clicks, sales)
- อย่าทำ social commerce โดยไม่มี trust-building content ก่อน
- ตรวจ algorithm changes ทุกไตรมาส — platform เปลี่ยนบ่อยมาก
