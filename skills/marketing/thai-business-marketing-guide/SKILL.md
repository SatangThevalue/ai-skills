# คู่มือการตลาดและธุรกิจในประเทศไทย (ทุกประเภท)

Skill นี้รวบรวมกรอบกลยุทธ์การตลาดและการวิเคราะห์ธุรกิจในบริบทประเทศไทย อิงจากงานวิจัยวารสาร ThaiJO (2025), สมาคมการตลาดแห่งประเทศไทย (MAT), DEPA, Krungsri Research, SCB EIC, และ OSMEP

ไม่ครอบคลุมการวิเคราะห์หลักทรัพย์, กฎหมายภาษี (ดู `thai-finance-legal-guide`), หรือการลงทะเบียนบริษัท (ดู `thai-business-registration`)

## When to Use

- วางแผนกลยุทธ์การตลาดสำหรับธุรกิจในไทย
- วิเคราะห์และเปรียบเทียบธุรกิจแต่ละประเภท
- ต้องการ Framework วัดผลการตลาดดิจิทัล
- ออกแบบแผนการตลาดสำหรับ SME งบจำกัด
- ค้นหาแหล่งข้อมูลเพื่อรีเสิร์ชตลาดไทย

## Prerequisites

- เข้าใจบริบทธุรกิจของตัวเอง (ขนาด / ประเภท / กลุ่มเป้าหมาย)
- ไม่ต้องใช้ API หรือ CLI ใด — เป็น Knowledge Skill

## Quick Reference

|-----------|---------|

## Procedure

---

### Phase 1 — วิเคราะห์ธุรกิจและบริบทตลาด

#### 1.1 ประเมินขนาดธุรกิจ

| ขนาด | นิยาม (OSMEP) | ลักษณะเฉพาะ |
|------|--------------|------------|
| **Micro** | พนักงาน <5 คน / รายได้ <1.8 ล้านบาท/ปี | เน้น Organic + LINE OA + Facebook |
| **S — Small** | พนักงาน 5-50 คน / รายได้ <100 ล้านบาท/ปี | เริ่มใช้ Paid Ads + Analytics |
| **M — Medium** | พนักงาน 50-200 คน / รายได้ <500 ล้านบาท/ปี | CRM + Multi-platform + Influencer |
| **L — Large** | พนักงาน >200 คน | Full-service + Brand Building |

#### 1.2 วิเคราะห์ประเภทธุรกิจตาม ROI ดิจิทัล (อ้างอิง: ThaiJO Research 2025, n=250)

| ประเภทธุรกิจ | ROI เฉลี่ย | กลยุทธ์ที่เหมาะ | ข้อควรระวัง |
|------------|-----------|----------------|------------|
| **เสื้อผ้า/แฟชั่น** | 24.7% (สูงสุด) | Visual Content + Micro-influencer + TikTok Shop | Micro-trend เปลี่ยนรายสัปดาห์ ต้อง Agile Inventory |
| **อาหาร/เครื่องดื่ม** | 21.3% | Community Building + Loyalty Program + Food Delivery | แข่งขันสูง ค่าส่งกระทบ Conversion |
| **สินค้าใช้ในบ้าน** | 19.8% (ต่ำสุด) | AR/Virtual Try-on + Google Shopping + รีวิวเชิงลึก | ลูกค้าต้องการสัมผัสก่อนซื้อ |
| **สุขภาพ/ความงาม** | สูง (Wellness trend 94%) | Clinical Content + Dermatologist Endorsement + K-beauty | FDA/อย. ต้องระวังคำโฆษณา (ดู `thai-content-compliance`) |
| **B2B/บริการธุรกิจ** | ขึ้นกับ Cycle | LinkedIn + Google Search + White Paper + Case Study | Sales Cycle ยาว ต้องลงทุน Content ระยะยาว |
| **ท่องเที่ยว** | 65% ผู้บริหารเชื่อมั่น | Quality Tourism + UGC + TikTok + Google Maps | ขึ้นกับ Seasonality และเศรษฐกิจโลก |

#### 1.3 กำหนด Ideal Customer Profile (ICP)

ไม่ใช่แค่ "อายุ 25–35 รายได้ปานกลาง" ต้องรู้:
- **ปัญหาที่เขาเผชิญ** — เขาเจ็บปวดตรงไหน?
- **คำค้นหาที่ใช้** — Google ด้วยคำว่าอะไร?
- **ปัจจัยตัดสินใจ** — ราคา / รีวิว / แบรนด์ / ความสะดวก?
- **จุดที่เขาอยู่ใน Journey** — รู้จักปัญหา / กำลังเปรียบเทียบ / พร้อมซื้อ?

### Phase 2 — วางกลยุทธ์ตาม ABCD Framework (MAT Thailand 2026)

| ตัวอักษร | หลักการ | การปฏิบัติ |
|---------|---------|-----------|
| **A** | Analytical Thinking + AI | ใช้ AI วิเคราะห์ Data → มนุษย์ตัดสินใจกลยุทธ์ |
| **B** | Balance Budget | งบแต่ละบาทต้องสร้างกำไร — รู้จังหวะ เร็ว/ช้า หนัก/เบา |
| **C** | Creativity under Constraints | "Drama Quality" ไม่ใช่ "Drama ไร้สาระ" |
| **D** | Data with Purpose | เก็บข้อมูลอย่างมีเป้าหมาย (Garbage in → Garbage out) |

### Phase 3 — จัดสรรงบ D-MROI Model (วารสาร ThaiJO 2025)

**สูตรจัดสรรงบ 70:20:10**
```
70% → Direct Advertising (Meta Ads, Google Ads, TikTok Ads)
20% → Analytics Tools + CRM (HubSpot, LINE OA Analytics)
10% → Experimentation (ทดลองช่องทางใหม่, A/B Test)
```

**จำนวนแพลตฟอร์มที่เหมาะสม**
- 3–4 แพลตฟอร์ม = ROI เฉลี่ย **24.2%**
- 5+ แพลตฟอร์ม = ROI เฉลี่ย **18.7%**
- ⚠️ มากแพลตฟอร์ม ≠ ROI ดีขึ้น — เพิ่มต้นทุนบริหาร

**งบต่อเดือน (แนะนำสำหรับ SME ไทย 2026)**

| งบ/เดือน | สิ่งที่ทำได้ |
|---------|------------|
| ฿25,000–50,000 | Social Media + Basic Ads |
| ฿50,000–100,000 | + SEO + Content Marketing |
| ฿100,000–150,000+ | Full-service + Influencer |

### Phase 4 — Thai Path-to-Purchase (Customer Journey ไทย)

เส้นทางที่ไม่เหมือนตะวันตก:
```
TikTok/Reels (Discovery)
  → LINE OA Chat (สอบถาม)
  → Shopee/Lazada (เปรียบเทียบราคา)
  → หน้าร้าน/ลองสินค้า (Validation)
  → Instagram (Reinforcement/แชร์)
```

**ช่องทางตามกลุ่มลูกค้า**

| กลุ่ม | พฤติกรรม | ช่องทางหลัก |
|------|---------|------------|
| B2B / ผู้บริหาร | ค้นเชิงลึกก่อนตัดสินใจ | Google Search + LinkedIn + บทความ |
| ผู้บริโภคทั่วไป | ตัดสินใจจากภาพ + รีวิว | Facebook + TikTok + Instagram |
| Local/ใกล้บ้าน | ค้นตามพื้นที่ | Google Maps + LINE OA |
| ลูกค้าเดิม (Retention) | ต้องการความสัมพันธ์ต่อเนื่อง | LINE + Email + Retargeting |

### Phase 5 — สร้าง Content 3-Stage Funnel

| Stage | เป้าหมาย | ตัวอย่าง |
|-------|---------|---------|
| **Awareness** | ทำให้รู้จักปัญหา | "ทำไม SME ยิง Ads แล้วไม่ได้ผล" |
| **Consideration** | ช่วยเปรียบเทียบ | "จ้างเอเจนซี่ vs ทำเอง — ต่างกันอย่างไร" |
| **Conversion** | แสดงหลักฐานผลลัพธ์ | Case Study + Reviews + CTA ชัดเจน |

**ข้อควรระวัง:** ถ้า Content มีแค่ Stage เดียว = พลาดลูกค้าอีก 2 กลุ่ม

**Video Strategy งบจำกัด**
- Hook ใน **3 วินาทีแรก** (คำถาม/ปัญหา/สิ่งน่าสนใจ)
- Short-form ≤60 วินาที = Discovery / Long-form = Engagement
- Repurpose: 1 คลิปยาว → 3-5 คลิปสั้น + Quote Cards + Stories
- โพสต์ **3-5 ครั้ง/สัปดาห์** (TikTok, Reels, YouTube Shorts)

### Phase 6 — วัดผลและปรับกลยุทธ์

**KPI ที่วัดผลได้จริง (ไม่ใช่ Vanity Metrics)**

| ❌ ห้ามใช้เป็นเป้าหลัก | ✅ ใช้วัดผลจริง |
|----------------------|---------------|
| Like, Reach, Follower | Cost Per Lead (CPL) |
| Impressions | Conversion Rate |
| Video Views | Revenue Attribution |

**Correlation กับ ROI (วิจัยไทย n=250)**
- งบการตลาดดิจิทัล → ROI: r = **0.642**
- การใช้ Analytics Tools → ROI: r = **0.598**
- ความถี่ในการโพสต์ → ROI: r = 0.443
- จำนวนแพลตฟอร์ม → ยอดขาย: r = 0.371

## ข้อแนะนำตามสถานการณ์เศรษฐกิจไทย 2026

- GDP คาดโต **0.9%** (MAT Survey, ต่ำสุดในรอบประวัติศาสตร์)
- **56%** ของผู้บริหารมองว่าปี 2026 "ยากที่สุด"
- งบโฆษณาดิจิทัลไทยคาดหดตัว **-32,145 ล้านบาท** (ติดลบครั้งแรกใน 14 ปี)
- **3P Priority ปี 2026:** Profit → People → Planet (จากเดิม People → Planet → Profit)

**อุตสาหกรรมที่เติบโตได้แม้เศรษฐกิจชะลอ:**
1. Health & Wellness (94% ผู้บริหารเชื่อมั่น)
2. Agri & Biotech (87%)
3. Quality Tourism (65%)

## Pitfalls

- **อย่าพึ่งพา Marketplace เพียงอย่างเดียว** — Shopee/Lazada/TikTok Shop ให้ยอดขายแต่ข้อมูลลูกค้าไม่ใช่ของคุณ ต้องสร้าง Own Channel ควบคู่ (LINE OA, เว็บไซต์)
- **อย่าวัดผลจาก Vanity Metrics** — Like/Follower ไม่ใช่กำไร ใช้ CPL และ Conversion Rate แทน
- **อย่าใช้ >4 แพลตฟอร์มพร้อมกัน** — ROI ลดลง เพราะต้นทุนบริหารสูงขึ้น
- **ระวัง Content ที่ไม่มี CTA** — ทุก Content ต้องมี "ขั้นตอนถัดไป" สำหรับลูกค้า
- **ระวังคำโฆษณาสินค้าสุขภาพ/ความงาม** — ต้องผ่าน อย./สคบ. (ดู `thai-content-compliance`)
- **Thai Path-to-Purchase ≠ Western Journey** — อย่านำ Framework ต่างประเทศมาใช้ตรงๆ

## Verification

Skill ทำงานถูกต้องเมื่อ:
- สามารถระบุ ROI เฉลี่ยตามประเภทธุรกิจได้ (F\u0026B, Fashion, Household)
- สามารถแนะนำ Channel ที่เหมาะกับกลุ่มลูกค้าแต่ละประเภท
- สามารถจัดสรรงบตามสูตร 70:20:10 พร้อมเหตุผล
- อ้างอิงแหล่งข้อมูลไทยได้อย่างน้อย 3 แหล่ง
