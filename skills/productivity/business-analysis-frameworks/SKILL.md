# Business Analysis Frameworks (MBA Strategy & Thai Adaptation)

## Overview
สกิลนี้รวบรวมและสรุปกรอบแนวคิดเชิงกลยุทธ์ระดับโลก (MBA Frameworks) สำหรับการวิเคราะห์ธุรกิจ การประเมินจุดอ่อนจุดแข็ง การสร้างแผนการเติบโต และการจัดการองค์กร พร้อมแนวทางการนำมาประยุกต์ใช้ในบริบทประเทศไทย (Thai Ecosystem) ทั้งด้านพฤติกรรมผู้บริโภค กฎหมาย และแพลตฟอร์มดิจิทัลยอดนิยม เช่น LINE/TikTok/Shopee

---

## When to Use
* **วิเคราะห์กลยุทธ์ธุรกิจและสภาพแวดล้อมการแข่งขัน** (Market Entry, Launching New Product)
* **ประเมินจุดแข็ง-จุดอ่อน และการหาขีดความสามารถการแข่งขัน** (Internal Audit, SWOT/VRIO)
* **วางแผนขยายกิจการและเลือกเส้นทางการเติบโต** (Growth Strategy, BCG Matrix, Ansoff Matrix)
* **ปรับโครงสร้างองค์กรและการจัดทัพทีมงาน** (Restructuring, McKinsey 7S)
* **วิเคราะห์เพื่อนำไปต่อยอดใช้ในบริบทตลาดไทย** (Thailand Adaptation)

---

## 1. SWOT & TOWS Matrix (การประเมินและการแปลงไปสู่กลยุทธ์ปฏิบัติ)

SWOT เป็นเครื่องมือประเมินสถานะปัจจุบัน แต่ **TOWS Matrix** คือเครื่องมือในการเปลี่ยนข้อมูลเหล่านั้นให้กลายเป็น **กลยุทธ์เชิงรุกและรับ (Actionable Strategy)**

```
                  ┌───────────────────────────┬───────────────────────────┐
                  │      Opportunities (O)    │        Threats (T)        │
                  │  (โอกาสภายนอกตลาดไทย)     │  (อุปสรรค/ความเสี่ยง)      │
┌─────────────────┼───────────────────────────┼───────────────────────────┤
│  Strengths (S)  │       SO Strategy         │       ST Strategy         │
│  (จุดแข็งภายใน) │  (เชิงรุก: ใช้จุดแข็งล่าโอกาส) │  (เชิงป้องกัน: ใช้จุดแข็งสู้ภัย) │
├─────────────────┼───────────────────────────┼───────────────────────────┤
│ Weaknesses (W)  │       WO Strategy         │       WT Strategy         │
│  (จุดอ่อนภายใน) │  (เชิงพัฒนา: แก้จุดอ่อนชิงโอกาส) │  (เชิงตั้งรับ: เลี่ยง/ปิดจุดอ่อน)│
└─────────────────┴───────────────────────────┴───────────────────────────┘
```

### การประยุกต์ใช้ในไทย (Thailand Context):
* **S (จุดแข็ง):** บริการหลังการขายที่รวดเร็ว (High-touch customer service) ความสามารถในการทำตลาดผ่าน Social Commerce การเข้าถึง Key Opinion Leaders (KOLs)
* **W (จุดอ่อน):** ปัญหาต้นทุนขนส่ง (Logistics cost in upcountry) การพึ่งพาแพลตฟอร์มต่างชาติมากเกินไป (e.g., ค่าโฆษณา Facebook/TikTok ที่สูงขึ้น)
* **O (โอกาส):** นโยบายผลักดัน Soft Power, การขยายตัวของสังคมสูงวัย (Silver Age Market), การใช้งาน LINE Official Account (LINE OA) และระบบ PromptPay ที่รวดเร็ว
* **T (อุปสรรค):** อัตราเงินเฟ้อ กำลังซื้อระดับล่างชะลอตัว กฎหมายภาษี e-Service และพ.ร.บ.คุ้มครองข้อมูลส่วนบุคคล (PDPA) ที่เข้มงวด

## 2. Growth Paths (การสร้างเส้นทางการเติบโต)

### A. Ansoff Matrix (ทางเลือกการเติบโต 4 ทิศทาง)
ใช้สำหรับเลือกว่าองค์กรจะเติบโตอย่างไร โดยวิเคราะห์ผ่าน **ผลิตภัณฑ์ (Products)** และ **ตลาด (Markets)**

1. **Market Penetration (เจาะตลาดเดิม - สินค้าเดิม):** เพิ่มยอดขายจากลูกค้าเดิมในไทย (เช่น การใช้ระบบ CRM, LINE OA Broadcast)
2. **Market Development (พัฒนาตลาดใหม่ - สินค้าเดิม):** ขยายไปยังต่างจังหวัด (Cross-border / Upcountry) หรือขยายไปกลุ่มประเทศ CLMV (Cambodia, Laos, Myanmar, Vietnam)
3. **Product Development (พัฒนาสินค้าใหม่ - ตลาดเดิม):** ออกสินค้าใหม่ขายกลุ่มลูกค้าเดิม (เช่น แบรนด์เครื่องสำอางไทยหันมาทำอาหารเสริม)
4. **Diversification (สร้างความหลากหลาย - สินค้าใหม่ ตลาดใหม่):** ข้ามอุตสาหกรรม (เช่น ธุรกิจอสังหาริมทรัพย์ไทย หันมาลงทุนในด้าน AI Agent/Tech Automation)

### B. BCG Matrix (การจัดสรรทรัพยากรในพอร์ตโฟลิโอ)
แบ่งกลุ่มธุรกิจ/สินค้าตาม **อัตราการเติบโตของตลาด (Market Growth Rate)** และ **ส่วนแบ่งตลาดสัมพัทธ์ (Relative Market Share)**

* **Stars (ดาวรุ่ง - โตสูง แชร์สูง):** ต้องลงทุนเพิ่มเพื่อรักษาตำแหน่ง (เช่น สินค้าเพื่อสุขภาพ/ออร์แกนิกในไทย)
* **Cash Cows (แม่โค - โตต่ำ แชร์สูง):** สร้างเงินสดต่อเนื่อง (เช่น สินค้าอุปโภคบริโภคหลัก) นำเงินสดมาเลี้ยงกลุ่ม Stars และ Question Marks
* **Question Marks (เครื่องหมายคำถาม - โตสูง แชร์ต่ำ):** มีโอกาสโตแต่ต้องตัดสินใจว่าจะลงทุนสู้ หรือตัดใจ (เช่น บริการ Web3/AI SaaS ในตลาดไทย)
* **Dogs (สุนัข - โตต่ำ แชร์ต่ำ):** ควรลดการลงทุน หรือปิดตัว (Divest)

### C. Growth Loops (ลูปการเติบโตแบบยั่งยืนแทน Marketing Funnel)
แทนที่จะหาลูกค้าใหม่ด้วยการยิงแอดตลอดเวลา (Linear Acquisition) MBA ยุคใหม่เน้นสร้าง **Growth Loops** ที่ผลลัพธ์ของลูปจะป้อนกลับเข้ามาเป็นตัวขับเคลื่อนรอบใหม่:
* **Product-led Loop:** ผู้ใช้ใช้ระบบ → สร้างผลลัพธ์แชร์ให้ผู้อื่น → ผู้ใช้รายใหม่เข้ามาใช้งาน (เช่น การแชร์สลิป PromptPay, ลิงก์ LINE LIFF)
* **User-Generated Content Loop:** ผู้ใช้รีวิวบน TikTok/Lemon8 → คนอื่นสนใจค้นหา → สมัครใช้งาน/ซื้อสินค้า → เขียนรีวิวใหม่

## 3. Industry & Competitor Analysis (การวิเคราะห์สภาพการแข่งขัน)

### A. Porter's Five Forces (แรงกดดันการแข่งขัน 5 ด้าน)
ช่วยประเมินความน่าดึงดูดใจของอุตสาหกรรมในไทย:

1. **Threat of New Entrants (ผู้เล่นใหม่):** ตลาดไทยมีทุนใหญ่หนุนหลังง่าย หากไม่มีสิทธิบัตรหรือเครือข่ายพันธมิตร (Barriers to Entry) จะโดนเลียนแบบเร็ว
2. **Threat of Substitutes (สินค้าทดแทน):** สินค้าราคาถูกจากจีนผ่านแพลตฟอร์ม Cross-border e-Commerce
3. **Bargaining Power of Buyers (อำนาจต่อรองของลูกค้า):** ลูกค้าไทยมีความภักดีต่อแบรนด์ (Brand Loyalty) ค่อนข้างต่ำ เปลี่ยนใจง่ายเมื่อมีโปรโมชั่นหรือโค้ดส่วนลด (Price-sensitive)
4. **Bargaining Power of Suppliers (อำนาจต่อรองของซัพพลายเออร์):** หากต้องนำเข้าวัตถุดิบ หรือใช้ Cloud Provider รายใหญ่
5. **Rivalry Among Existing Competitors (การแข่งขันท่ามกลางคู่แข่งเดิม):** สงครามราคา (Price War) ในตลาด E-commerce/Platform Delivery

### B. VRIO Framework (การหาข้อได้เปรียบในการแข่งขันที่ยั่งยืน)
วิเคราะห์จากทรัพยากร/ความสามารถภายในองค์กรผ่านคำถาม 4 ข้อ:
1. **Value (มีมูลค่าหรือไม่):** ช่วยลดต้นทุนหรือเพิ่มรายได้ในบริบทไทยได้จริงหรือไม่?
2. **Rarity (หายากหรือไม่):** คู่แข่งในไทยรายอื่นมีทรัพยากรนี้หรือไม่?
3. **Imitability (ลอกเลียนแบบยากหรือไม่):** มีสิทธิบัตร เครือข่ายเฉพาะตัว หรือวัฒนธรรมองค์กรที่ลอกยากหรือไม่?
4. **Organization (พร้อมใช้ประโยชน์หรือไม่):** ระบบและโครงสร้างองค์กรรองรับการนำไปใช้สร้างมูลค่าหรือไม่?

## 4. Internal Alignment (การจัดทัพองค์กรภายใน)

### McKinsey 7S Framework
เมื่อปรับเปลี่ยนกลยุทธ์ (Strategy) องค์กรต้องปรับสมดุลขององค์ประกอบทั้ง 7 ด้านที่เชื่อมโยงกัน:

* **Hard S (ปรับเปลี่ยนได้โดยตรงผ่านโครงสร้าง):**
  1. **Strategy:** กลยุทธ์การแข่งขัน
  2. **Structure:** โครงสร้างการแบ่งสายงาน (เช่น การตั้งทีม Agile/Cross-functional ในองค์กรไทย)
  3. **Systems:** ระบบงานและขั้นตอนปฏิบัติการ (เช่น ERP, Automated CRM, Data Architecture)
* **Soft S (ขับเคลื่อนด้วยวัฒนธรรมและคน ต้องใช้เวลาปรับ):**
  4. **Shared Values:** ค่านิยมร่วม (เป็นศูนย์กลางที่ยึดโยงอีก 6 ตัวไว้ด้วยกัน)
  5. **Skills:** ทักษะความสามารถของพนักงาน (ความเข้าใจเรื่อง Digital Literacy / Data Analytics)
  6. **Staff:** ทีมงานและการสรรหาบุคคลากร (Talent Management ในตลาดไทยที่ขาดแคลนด้าน Tech)
  7. **Style:** รูปแบบการบริหารจัดการของผู้นำ (การลดขั้นตอนความเป็นเจ้านาย-ลูกน้อง หรือ Hierarchy ลง เพื่อความเร็ว)

## Common Pitfalls
1. **SWOT แบบคิดเอาเอง (Subjective Bias):** วิเคราะห์ SWOT โดยไม่มีตัวเลขหรือข้อมูลตลาดมารองรับ แก้ไขโดยใช้ข้อมูลสถิติจริง ปริมาณการค้นหา คีย์เวิร์ด หรือข้อมูลคู่แข่งประกอบ
2. **ทำ SWOT แล้วจบที่กระดาษ:** ไม่แปลง SWOT เป็น TOWS Matrix ทำให้ออกแผนปฏิบัติการจริงไม่ได้
3. **ใช้ Framework ทั่วโลกตรงๆ โดยไม่ปรับบริบทไทย:** เช่น คิดแผนการตลาดสหรัฐฯ ที่เน้น Email Marketing แต่สำหรับตลาดไทยต้องเปลี่ยนมาเน้น LINE OA / Chat Commerce / TikTok Live
4. **ลืมวิเคราะห์อำนาจของแพลตฟอร์มยักษ์ใหญ่:** ในข้อ Porter's Five Forces ของไทย ต้องพิจารณา Platform Fee และอัลกอริทึมของแพลตฟอร์ม (e.g. Meta, Shopee, TikTok) เป็น Supplier/Intermediary ที่มีผลกระทบสูงมาก

## Verification Checklist
- [ ] มีการทำ TOWS Matrix ควบคู่กับการวิเคราะห์ SWOT เสนอแนะแนวกลยุทธ์ปฏิบัติจริง
- [ ] การทำ Ansoff Matrix ต้องระบุวิธีการเจาะตลาดในไทยอย่างชัดเจน (เช่น ใช้แพลตฟอร์มใด ขยายกลุ่มลูกค้าอย่างไร)
- [ ] มีการประเมิน VRIO ครบทั้ง 4 ปัจจัย และสรุปขีดความสามารถการแข่งขัน (Sustained Competitive Advantage)
- [ ] แผนธุรกิจมีการตรวจสอบความสอดคล้องกับพฤติกรรมผู้บริโภคชาวไทย (เช่น พฤติกรรมการชอบแชทถามก่อนซื้อ, การสแกนจ่ายเงิน QR Code)
